# Instagram Data Analysis Project

## Project Overview
The main objective of this project was to analyze Instagram data to understand the factors that influence the reach and engagement of posts.

## Key Insights
* **Engagement:** We observed a strong, positive correlation between Impressions and Likes.
* **Reach Sources:** The analysis indicates that the majority of the reach is driven by the home feed.
* **Predictive Model:** We utilized a Linear Regression model, which successfully predicted reach with an accuracy of approximately 72%.

## Conclusion
To maximize engagement, it is essential to focus on content that encourages 'Saves' and time posting based on the insights gained from the Reach sources.
